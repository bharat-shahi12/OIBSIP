Online Train Reservation System

A simple Java console-based train reservation system that allows users to book, cancel, and view tickets. Each ticket gets a unique PNR, and the system includes a login authentication feature. Built using Java OOP and HashMap collections, it demonstrates core programming concepts while providing a functional, menu-driven interface.

Technologies: Java, Scanner, HashMap, OOP

Features:

1.Secure login

2.Book tickets with auto-generated PNR

3.Cancel tickets using PNR

View all reservations

This project is a foundation for further improvements, such as a GUI version, multiple train options, and persistent storage.
